# Admissions Reference Overview

Source basis: United Ceres College admissions reference guide and the current local admission-screening rules.

## Purpose

This file summarizes the working admissions screening approach used in the app:

- AI extracts evidence from applicant materials.
- The final result is determined by rule-based MER checks.
- Missing or unclear evidence should trigger manual review rather than guessed decisions.

## Core screening principles

- Do not let AI guess the final admission decision.
- Use documented academic and English benchmarks.
- Keep equivalency rules editable.
- Escalate edge cases to manual review or interview review.

## Common recommendation outcomes

- Eligible
- Not Eligible
- Requires Manual Review
- Requires Interview
- Requires English Placement Test
- Requires Additional Documents

## Operational note

These markdown files are intended as local editable alternatives and working references for admissions staff and future rule expansion.
