# Course MER Authoring Notes

This file supports the Courses and MER Settings page.

## Recommended course fields

- Course ID: stable machine-friendly slug
- Course Name: staff-facing title
- Intake: e.g. Monthly, Quarterly
- Minimum Age
- Academic MER criteria
- English MER criteria

## Authoring guidance

- Keep each criterion short and auditable.
- Separate academic, English, and age logic.
- Put descriptive wording in the course config, not hidden in code.
- Add new equivalencies to the shared rules file when a course relies on them.

## When to create a new course record

- New programme with different age threshold
- Different English benchmark
- Different experiential entry pathway
- Different academic equivalency logic
